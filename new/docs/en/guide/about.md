# About Talebook

Talebook is a personal book management system built on [Calibre](https://calibre-ebook.com/), simple yet feature-rich.

## Key Features

- **Modern UI**: Built with Nuxt 4 + Vue 3 + Vuetify, responsive design, zh/en i18n, light/dark theme
- **Online Reading**: Built-in [candle-reader](https://github.com/talebook/candle-reader), supports EPUB, PDF, MOBI, AZW3 and more
- **Smart Metadata**: Multi-source book info search (Douban, Baidu Baike, etc.) with AI auto-recognition
- **Multi-User**: Social login via QQ, Weibo, GitHub, fine-grained permissions
- **Online Book Sources**: Import Legado-style book sources, search and read web novels
- **OPDS**: Compatible with KyBooks and other reading apps
- **Kindle Delivery**: Manage multiple devices, one-click push with auto conversion
- **Desktop Client**: [Moke](https://github.com/talebook/moke) desktop app for Windows/macOS/Linux
