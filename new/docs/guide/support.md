# 支持我们

我们是开源免费项目，Talebook 的持续开发和维护离不开社区的支持。如果你觉得这个项目对你有帮助，欢迎通过以下几种方式支持我们：

## 点个 Star ⭐️

在 GitHub 上给 [Talebook](https://github.com/talebook/talebook) 点一个 Star。

## 爱发电

通过 [爱发电平台](https://afdian.com/@talebook) 赞助作者，持续优化，为爱充电！

## 成为贡献者

欢迎提交 Issue、PR，或者参与文档翻译和社区维护。

[![Contributors](https://contrib.rocks/image?repo=talebook/talebook)](https://github.com/talebook/talebook/graphs/contributors)

如果你愿意支持我们，非常感谢你，我们也会继续用更优秀的产品回应你的信任。
