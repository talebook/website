# 关于 Talebook

Talebook 是一个基于 [Calibre](https://calibre-ebook.com/) 的个人图书管理系统，简洁但功能丰富。

## 主要特性

- **美观的界面**：基于 Nuxt 4 + Vue 3 + Vuetify 构建，支持 PC/手机浏览，中英文多语言，白天/夜间主题切换
- **在线阅读**：内置 [candle-reader](https://github.com/talebook/candle-reader) 阅读器，支持 EPUB、PDF、MOBI、AZW3 等格式
- **智能元数据**：支持豆瓣、百度百科、新华书店、番茄小说等多源搜索导入，还可接入 AI 大模型自动识别
- **多用户支持**：支持 QQ、微博、GitHub 等社交登录，完善的权限管理
- **网络书库**：导入 Legado 风格书源，在线搜索阅读网络小说，一键保存到本地
- **OPDS 支持**：兼容 KyBooks 等阅读 APP，支持浏览和导入外部 OPDS 书库
- **Kindle 推送**：管理多个 Kindle 设备，一键推送，自动批量格式转换
- **桌面客户端**：配套 [Moke](https://github.com/talebook/moke) 桌面应用，跨平台支持 Windows/macOS/Linux
