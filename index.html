Tale Book

talebook是一个开源项目，目标是提供一个简洁但功能丰富的书籍管理系统。
它基于calibre项目构建，具备书籍管理、在线阅读与推送、用户管理、SSO登录、从百度/豆瓣拉取书籍信息等功能。

项目地址 查看演示

为爱书之人精心设计

基于Vuetify框架构建的页面，体验顺滑，功能丰富，为爱书之人所精心设计

丰富的特性

提供在线书站所具备的常见能力，包括书籍浏览与管理、在线阅读、邮件推送Kindle、书籍信息更新、快速安装、用户管理、SSO登录（QQ、微博、微信、Github等）、以及opds接口（读书APP连接）、强大的后台管理能力

快速配置

具备可视化快速安装配置界面，一键设置书籍站点信息

在线阅读

集成readium.js库，能够对书籍进行在线阅读

搜索书籍信息

可以从百度/豆瓣上同步书籍信息

丰富管理界面

支持社交网站SSO登录，以及邮箱注册用户能力，具备丰富的书库配置能力

立即使用

本项目提供有docker镜像，可以在群晖NAS等系统上快速安装启动。如果您的服务器上安装有docker程序，可以通过以下命令启动：

docker run -d --name talebook -p 8080:80 -v /localdata:/data talebook/talebook
